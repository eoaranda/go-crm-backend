API v1 Documentation

This is a RESTful API v1 for the the final project "CRM Backend project".

Method URL Description
GET /customers/{id} Getting a single customer through a /customers/{id} path
GET /customers Getting all customers through a the /customers path
POST /customers Creating a customer through a /customers path
PUT /customers/{id} Updating a customer through a /customers/{id} path
DELETE /customers/{id} Deleting a customer through a /customers/{id} path
